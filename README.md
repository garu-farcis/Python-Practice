# Python for Data Engineering 🐍

A structured, progressive practice repository covering Python 
fundamentals and data engineering libraries — built as part of 
an active upskilling roadmap.

## 🎯 Purpose

This repo documents hands-on Python practice with a focus on 
data engineering use cases: data manipulation, numerical computing, 
and data visualisation.

## 📚 Topics Covered

**Core Python**
- Data types, control flow, functions, error handling
- OOP principles: classes, inheritance, encapsulation

**Data Engineering Libraries**
- **Pandas** — DataFrame manipulation, cleaning, joins, groupby aggregations
- **NumPy** — Array operations, vectorised computation, linear algebra
- **Matplotlib** — Line charts, bar charts, scatter plots, subplots
